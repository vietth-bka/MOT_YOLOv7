# Multiple Object Tracking using YOLOv7


## Introduction
This repository was heavily inspired from [YOLOv7](https://github.com/WongKinYiu/yolov7). For tracking in 360 thermal camera, we used YOLOv7 as the main detector.


## Highlights
- We deployed a real-time multi-object tracking algorithm in a 360-degree thermal camera using YOLOv7
combined with backbone ResNet18 in Re-Identification block.
- To mitigate the issue of small and fast-moving objects, we abandoned IOU distance in favor of Mahalanobis Distance when computing motion loss, 
appearance loss was computed on the basis of Cosine distance between embeddings generated by the backbone of the Re-Id block.
- Devised a contraint on the size of objects when combining the two informations (motion and appearance).

## Inference
```shell
python demo_track.py —weights "yolov7-path" —source "your-input-sequence"
```
